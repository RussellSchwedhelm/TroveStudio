> "Less is more. A fashion-focused selection of timeless jewelry with a casual elegance."

Trôve Studio is a premium, minimalist e-commerce storefront designed for a luxury jewelry brand. Inspired by the sleek, high-end aesthetics of [LIE Studio](https://lie-studio.dk/), this project showcases a sophisticated blend of typography, generous white space, and interactive elements to create a seamless shopping experience.

---

## ✨ Key Features

- **💎 Minimalist Aesthetic**: A clean, "quiet luxury" design focused on high-quality imagery and elegant typography.
- **📱 Fully Responsive**: Optimized for all devices—from large desktop monitors to tablets and smartphones.
- **🖼️ Interactive Hero Grid**: A dynamic 3-column immersive entry point with smooth hover scaling and overlays.
- **🎨 Modern Design Tokens**: Utilizes a central CSS variable system for easy branding and consistency.
- **✨ Sticky Header**: A smart navigation bar that transforms upon scrolling to maintain focus on content.
- **🛍️ Product Showcase**: A clean, balanced grid system for highlighting new arrivals and featured collections.
- **📧 Newsletter Integration**: Minimalist subscription form for customer engagement.
- **🛠️ Optimized Assets**: High-resolution imagery with performance-focused loading.

---

## 🚀 Tech Stack

- **Core**: HTML5, Vanilla CSS3, Modern JavaScript (ES6+).
- **Icons**: [Lucide Icons](https://lucide.dev/) for crisp, scalable vector graphics.
- **Typography**: [Google Fonts](https://fonts.google.com/)
  - *Playfair Display*: For a timeless, editorial feel.
  - *Inter*: For modern, legible interface text.
- **Layout**: CSS Grid and Flexbox for precise, responsive alignment.

## 🎨 Design Philosophy

Trôve Studio is built on the principle of **Functional Minimalism**. Every element serves a purpose, ensuring that the jewelry remains the focal point. The use of a monochrome palette with subtle borders and shadows creates a sense of depth without clutter.

- **Typography**: The pairing of *Playfair Display* (Serif) for headings and *Inter* (Sans-serif) for body text mirrors high-end fashion editorials.
- **Transitions**: Every interaction, from link hovers to image scaling, uses soft `ease-in-out` transitions to evoke a feeling of luxury.
